novaSignature

Prerequisites

  • In Admin console give access to service account
  • Org Unit name should not contain apostrophe or any other special characters
  • Make sure you are logged in as domain admin
  • Upload images that you will be using in the signature. (Optional)

Main pages in novaSignature

  • Dashboard: Showing statistics of the tool, failure/success in last 7 days. Searchable error logs. Also track progress of Rule application (Status: In Progress, Completed). Dashboard is refreshed from time to time, please wait for few moments to see the changes.
  • Images: Upload your images in GCS, that can later be used to insert in signature.
  • Signatures: Signature templates that can be applied.
  • Rules: Rules dictate in which org units signature will be applied. All the updates will happen according to rules, individual user signature update is not supported.

Steps to use the tool

Create/edit signature templates:

This is the page to create/edit signatures. Please create/edit signatures before moving on to Rules page.

  • You can insert all kind of attributes (name, office number, mobile number) and formatting that will be applied to the signature.
  • You can insert image from GCS or paste the link.
  • You can insert youtube link that will also show preview (make sure to use shareable link).
  • Once you save the signature you can move to rules page.
  • Also, from Signature page you can see preview of how signature is going to look.

Create/edit Rules templates:

Please create/edit/review signature before this step. This is the page to apply signature to an org unit.

  • You can either apply existing rule or create/edit.
  • You have to select Org Unit for which you want signature to be applied. You can select either one or more org units.
  • Next, select the signature template that you want to apply.
  • After saving or creating a rule you have to hit apply to start the process.
  • Track progress on Dashboard. It will show if the process is ‘In Progress’ or it is ‘Completed’. On an average, 10 users are processed per second.So,you can expect completion time accordingly.
